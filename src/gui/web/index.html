    Activity

    Complete causal log. Every search records its routing decision, matched entity, and per-fact scores — cosine similarity, RRF fusion, ACT-R activation (recency + frequency decay), and final ranking. Every ingest records classification, per-fact AUDM verdicts with the similarity that drove them, and entity links. Persisted and replayable.

      Engine

      The LLM core. When the managed-session engine is on, Sigil keeps warm claude workers alive in tmux (one sigil-<worker> session each) to amortize agentic cold-starts across ingest + expansion calls. When off, every call uses the one-shot claude -p path. Live worker state below; per-task dispatch/result/fallback events stream into Activity (filter: Engine).
